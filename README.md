# Skill-Based Job Recommendation System

## Overview
This project automates the job search process by matching user skills with relevant job postings using **NLP, LLMs, and web scraping**.

- Users upload their resumes (PDF/DOCX).  
- System extracts key skills using LLM.  
- Scrapes job portals (e.g., Naukri.com) to find matching jobs.  
- Ranks and displays top recommendations on a dashboard.

## Technologies Used
- Python
- Django 
- HTML/CSS
- Selenium
- Natural language processing(NLP)
- Large Language Models (LLM)
- Pandas, Numpy

## Features
- Automatic skill extraction from resumes
- Real-time job scraping and matching
- Personalized job ranking
- User-friendly dashboard
